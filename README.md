# Headphones - Implement a Design from Scratch

This project is a fully responsive web page implemented from scratch using only HTML and CSS — no external frameworks, no JavaScript. The goal is to faithfully replicate a professional UI/UX design created by Nicolas Philippot, matching every detail of the Figma mockup including layout, typography, spacing, and color. The page features a hero header, feature highlights, results showcase, a contact form, and a footer, all built with semantic HTML and modern CSS techniques.

The web page is fully responsive: on screens 480px wide or less, the layout automatically switches to a mobile-friendly version. Interactive elements follow the design spec — links highlight in `#FF6565` on hover/active, buttons reduce to `opacity: 0.9` on hover, and the main content area is capped at 1000px and centered in the viewport.

![Headphones Web Page Design Preview](https://intranet-projects-files.s3.amazonaws.com/holbertonschool-webstack/622/01_headphones_first_desktop.jpg)

## Technologies

- HTML5 (semantic markup)
- CSS3 (Flexbox, media queries, custom properties)

## Responsive Breakpoint

- Mobile layout activates at **480px** and below
